# [Kth Node From End of List LCCI][title]

## Description

实现一种算法，找出单向链表中倒数第 k 个节点。返回该节点的值。

**注意：** 本题相对原题稍作改动

**示例：**
            **输入：** 1->2->3->4->5 和 _k_ = 2    **输出：** 4

**说明：**

给定的 _k_  保证是有效的。


**Tags:** Linked List, Two Pointers

**Difficulty:** Easy
