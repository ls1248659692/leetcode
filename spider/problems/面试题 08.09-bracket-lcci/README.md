# [Bracket LCCI][title]

## Description

括号。设计一种算法，打印n对括号的所有合法的（例如，开闭一一对应）组合。

说明：解集不能包含重复的子集。

例如，给出 n = 3，生成结果为：
            [      "((()))",      "(()())",      "(())()",      "()(())",      "()()()"    ]    


**Tags:** String, Dynamic Programming, Backtracking

**Difficulty:** Medium
