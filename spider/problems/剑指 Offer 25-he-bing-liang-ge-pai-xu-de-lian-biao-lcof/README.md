# [合并两个排序的链表  LCOF][title]

## Description

输入两个递增排序的链表，合并这两个链表并使新链表中的节点仍然是递增排序的。

**示例1：**
            **输入：** 1->2->4, 1->3->4    **输出：** 1->1->2->3->4->4

**限制：**

`0 <= 链表长度 <= 1000`

注意：本题与主站 21 题相同：<https://leetcode-cn.com/problems/merge-two-sorted-lists/>


**Tags:** Recursion, Linked List

**Difficulty:** Easy
