# [Factor Combinations][title]

## Description

整数可以被看作是其因子的乘积。

例如：
            8 = 2 x 2 x 2;      = 2 x 4.

请实现一个函数，该函数接收一个整数 _n_  并返回该整数所有的因子组合。

**注意：**

  1. 你可以假定 _n_ 为永远为正数。
  2. 因子必须大于 1 并且小于 _n_ 。

**示例 1：**
            **输入:**1    **输出:** []    

**示例 2：**
            **输入:**37    **输出:** []

**示例 3：**
            **输入:**12    **输出:**    [      [2, 6],      [2, 2, 3],      [3, 4]    ]

**示例 4:**
            **输入:**32    **输出:**    [      [2, 16],      [2, 2, 8],      [2, 2, 2, 4],      [2, 2, 2, 2, 2],      [2, 4, 4],      [4, 8]    ]    


**Tags:** Array, Backtracking

**Difficulty:** Medium
